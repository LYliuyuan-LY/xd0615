# xd0615
学习笔记：
		1. 登录github.com
		2. 申请自己的账户
		3. 创建一个项目仓库
		4. 使用git clone命令，将项目克隆的本地
		5. 在本地编辑
		6. 使用git add 将修改后文件保存在本地仓库。
		7. 使用git commit -m命令设置注释
		8. 使用git push命令将本地项目更新提交到github。

三个需要自己搜索掌握的知识：
		1. 安装git工具
		2. 设置ssh-keygen，直接访问github。
		3. git的基本操作。
